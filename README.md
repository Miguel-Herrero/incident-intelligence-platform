# Incident Intelligence Platform

> An AI-powered system for automated incident diagnosis and root cause analysis, built with production-grade infrastructure.

## ğŸ¯ Project Vision

This platform demonstrates how to build reliable, observable, and cost-efficient AI systems for SRE/DevOps workflows. It combines:

- **AI Gateway** (Go): Multi-provider LLM routing, semantic caching, rate limiting, and cost tracking
- **Agent Orchestration** (Python + LangGraph): Multi-agent system for log analysis, metrics correlation, and runbook matching
- **RAG Pipeline**: Multi-source ingestion (logs, metrics, runbooks, code)
- **Evaluation Framework**: Automated evals with quality metrics and CI/CD integration
- **MCP Server**: Standards-based tool exposure for any compatible AI client
